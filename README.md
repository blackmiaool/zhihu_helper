# zhihu_helper
知乎增强油猴脚本

### 可选功能
* 复制时去除版权标注
* 去除外链跳转中转
* 民主自由标签
* 隐藏黑名单中用户的评论
* +1s/-1s

### chome下安装方法
* 安装[油猴脚本插件](https://chrome.google.com/webstore/detail/dhdgffkkebhmkfjojejmpbldmpobfkfo)
* 点击右上角插件图标-Dashboard-Utilities 
* 把 http://blackmiaool.com/zhihu_helper/index.js 这个地址粘贴到输入框,点import
* 点击install
* 进入知乎
